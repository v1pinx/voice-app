# Voice Transcriber

## Overview
Voice Transcriber is a android application that records audio and converts it into text in real-time. It utilizes the Gemini API for summarization and task extraction.

## Features
- Real-time speech-to-text conversion using GBoard
- Reset transcription
- User-friendly interface

## Technologies Used
- React Native
- Gemini API


## Future Enhancements
- Add support for multiple languages.
- Implement offline transcription.
- Improve UI/UX with animations and themes.
